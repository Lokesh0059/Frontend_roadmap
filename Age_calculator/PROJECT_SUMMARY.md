## ðŸ“ž Troubleshooting

**Problem:** Port 8080 already in use
**Solution:** Edit `webpack.config.js` line 13, change `port: 8080` to 3000 or another number

**Problem:** Styles not loading
**Solution:** Verify `import './style.css'` is in `src/index.js`

**Problem:** Datepicker not showing
**Solution:** Check Flatpickr CSS import: `import 'flatpickr/dist/flatpickr.min.css'`

**Problem:** npm install fails
**Solution:** Clear cache: `npm cache clean --force`, then `npm install`

For more help, see SETUP.md or README.md
